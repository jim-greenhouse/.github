# jim-greenhouse/.github

Этот репозиторий хранит **метаданные** организации GreenSense на GitHub:

- [`profile/README.md`](./profile/README.md) — публичный README, который отображается на главной странице организации [github.com/jim-greenhouse](https://github.com/jim-greenhouse).
- `profile/logo.svg` — логотип, на который ссылается README.

Сюда же можно складывать общие для всех репозиториев конфигурации:

- `.github/CODE_OF_CONDUCT.md`, `CONTRIBUTING.md`, `SECURITY.md`, `SUPPORT.md` — наследуются всеми репозиториями организации, в которых нет своих версий.
- `.github/ISSUE_TEMPLATE/`, `.github/PULL_REQUEST_TEMPLATE.md` — общие шаблоны issue и PR.
- `.github/FUNDING.yml` — кнопка «Sponsor».

> Этот репозиторий **не** содержит кода продукта. Код живёт в [`web-app`](https://github.com/jim-greenhouse/web-app), [`landing`](https://github.com/jim-greenhouse/landing), [`api`](https://github.com/jim-greenhouse/api), [`gateway`](https://github.com/jim-greenhouse/gateway), [`firmware`](https://github.com/jim-greenhouse/firmware) и других репозиториях организации.
